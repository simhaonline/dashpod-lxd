Secure connections to remote LXD servers are made using the LXD remote TLS protocol. Before adding a remote host, first copy the client certificate information and paste it into a file on your remote server such as dashpod.crt. Then import the certificate on your remote server by running the command: lxc config trust add dashpod.crt, replacing the filename.
